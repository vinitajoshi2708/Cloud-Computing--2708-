# Node.js App Deployment on Vercel

## Project Description
This project is a simple Node.js web application created using Express framework. The application runs on a local server and is deployed on Vercel for public access.

## Aim
To create and deploy a Node.js application on Vercel.

## Technologies Used
- Node.js
- Express.js
- GitHub
- Vercel

## Steps to Run Locally

1. Install Node.js  
2. Initialize project:
   npm init -y  

3. Install dependencies:
   npm install express  

4. Run the application:
   node index.js  

5. Open browser:
   http://localhost:8000
